# Gestion App

## Full Stack Web Application

This project is a full stack web application built using Java Spring Boot as the backend framework. The application is designed to help manage various tasks and operations efficiently.

### Features
- **User Authentication**: Secure login and registration process.
- **Task Management**: Create, update, and delete tasks.
- **RESTful API**: Interact with the application through REST endpoints.
- **Responsive Design**: User-friendly interface that works on various devices.

### Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Java Spring Boot
- **Database**: MySQL

### Getting Started
1. Clone the repository.
2. Install the necessary dependencies.
3. Configure application properties (e.g., database connection).
4. Run the application.

### Installation
To install the project, run:
```bash
# Clone the repository
git clone https://github.com/MarianoRama/gestion-app.git

# Navigate into the project directory
cd gestion-app

# Build the application
./mvnw clean install
```
